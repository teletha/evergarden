VioletEvergarden.DocumentDoclet

  • BaseDoclet
  • Doclet

Internal Doclet specifically for processing document and sample source files.

DO NOT USE THIS CLASS DIRECTLY. It is public only due to Doclet API limitations.

It identifies `*Manual.java` files to be treated as primary document pages and extracts code snippets referenced by `@see` tags in test methods (`*Test.java`) as examples (Doodles).

DocumentDoclet()

process(ClassInfoinfo)void

ClassInfoinfo

The ClassInfo of the type being processed.

Processes a type element found during the document scan. If it's a public class ending in "Manual", it's added to the main document list. Otherwise, it scans methods for @see tags pointing to examples and registers them as Doodle objects in the Letter.

complete()void

Called after processing all document/sample files. Triggers the building of the document navigation tree within the Letter.
