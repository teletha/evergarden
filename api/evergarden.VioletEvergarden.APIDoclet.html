Internal Doclet specifically for processing API source files.

DO NOT USE THIS CLASS DIRECTLY. It is public only due to Doclet API limitations.

It analyzes each type element encountered and registers its ClassInfo representation with the main Letter model.

process(ClassInfoinfo)void

ClassInfoinfo

The ClassInfo of the type being processed.

Processes a type element found during the API scan. Registers the corresponding ClassInfo with the Letter.

complete()void

Called after processing all API source files. Triggers the building of type relationships (inheritance, usage) within the Letter.
