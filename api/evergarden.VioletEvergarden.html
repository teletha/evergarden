Abstract base class for configuring and executing the Evergarden Javadoc generation process. This class provides a fluent API (using Icy annotation for implementation generation) to set up source directories, output location, classpath, project metadata, external documentation links, and finally trigger the documentation build process.

The documentation generation involves scanning API source code, document/sample source code, building an internal representation (likely via the Letter model), and then rendering the final static website (HTML, CSS, JS).
