Region

  • Record

Represents a specific region within a document, identified by a location (e.g., file path or URL) and an optional line range.

This class is typically used to refer to sections of source files or other line-based documents. If line numbers are unknown or not applicable, they will default to -1.

Region(Stringlocation)

Stringlocation

the document location (e.g., file path or URL)

Creates a Region with only a location. Line numbers will default to -1, indicating that they are unspecified.
